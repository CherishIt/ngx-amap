NGX-AMAP

ngx-amap 是为在Angular(ver >= 2.x)项目中便捷、高效地使用高德地图Javascript API而构建的组件集合

查看文档 Github

## 安装 ```bash $ npm install ngx-amap --save # or $ yarn add ngx-amap ```
## 使用 ```typescript import {{'{'}} NgxAmapModule } from 'ngx-amap'; @NgModule({{'{'}} imports: [ ..., NgxAmapModule.forRoot({{'{'}} apiKey: '你申请的key' }) ], ... }) export class AppModule {{'{'}} } ```
